搭建选择

发表于2022-11-17|更新于2023-02-23|hexo
|字数总计:732|阅读时长:2分钟|阅读量:

主要讲一下接触过的，像Hugo，Typecho这里不赘述了

Hexo

一般选择结合github使用，程序也是免费开源的，它借助Github提供的Github Page服务，可以轻松建立免费优质的静态网站。
hexo相对更简洁点，很容易弄明白每个文件是干什么的，可以魔改出自己的hexo，通过各种方法实现自己的要求。需要有一定的git基础。不过使用比较折腾，自由度高，有能力的情况下能高度DIY定制，需要node.js 和git 才能写博客。

node.js git 简单介绍

node.js
需要配置环境变量，以及npm配置,详细安装步骤可移步至此：
【Node.js安装及环境变量配置-哔哩哔哩】
node.js官网下载
判断是否安装成功 node -v

git

是一种版本控制工具，版本控制工具，可以分为本地版本控制系统、集中化的版本控制系统和分布式版本控制系统，git 属于分布式的版本控制系统。
官网直接安装，一直点击下一步
Git教程文本可参考：https://haicoder.net/git/git-repo.html

小结

优点
它可以没有服务器，甚至域名，像我这种白嫖党，可以试试。hexo只负责推送文章就行了，本地写好内容，git上传到仓库就行了。很安全，因为不用担心服务器会不会挂了，谁让你没后端呢，嘿嘿。如果换电脑的话，还要重新配置环境（目前我没尝试过，因为我没有换电脑的经济基础😔）
缺点
hexo由于没有后端,所以评论功能只能外接其他平台。（可以做但没必要，反正我自己的hexo就写文章就行了），总之高投入高回报。折腾起来吧😀

Word press

相对于静态hexo，不需要过多的折腾，像我这种花不了太多的时间，放在维护上，有什么问题，一般上网可查询解决。所以开箱即“食”😀(我的blog欢迎大家光临https://www.hlsiyy.love/)
W是一款动态网页，易于扩展拥有大量插件以及用户。
全球成百上千万的企业网站都是用WordPress系统搭建起来的。WordPress的流行要归功于它的功能丰富，也归功于开源的力量.WordPress使用PHP语言编写，数据库使用的是MySQL，是完全开源的软件方案，并且WordPress程序也是完全开源的。说白了就是简单易用。当然虚拟主机配置低的有的还带不动，访问速度慢等等问题

总结

H和W都有优缺点，综上根据自己情况，选择自己的方式搭建。当然如果你有”钞“能力，可忽略以上。毕竟能用钱解决的问题，都不是问题。
